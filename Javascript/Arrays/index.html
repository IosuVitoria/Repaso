Repaso de Arrays

Los siguientes ejercicios muestran casos sencillos de posibles situaciones que pueden ocurrir durante la manipulación de arrays.

Ejercicio 1

Obtener primer y último elemento de un array. Recuerda el uso de los corchetes y la medición de la longitud de un array. 

const numeros = [1,2,3,4,5,6,7,8,9];

Ejercicio 2

Manipulación de nombres o elementos de un array. Hora de repasar: pop, push, shift y unshift...

const nombres = ["Iosu", "Vero", "Aiur"];

Ejercicio 3

Buscar la posición de un número en un array. También funciona con strings, me refiero a indexOf.

const numeros2 = [5, 10, 15, 20, 25];

Ejercicio 4

Ordenar un array de forma ascendente y descendente. Sort() y Slice combinados para ordenar arrays. 

const numeros3 = [3, 1, 4, 1, 5, 9, 2, 6, 5];

Ejercicio 5

Multiplicar por 2 cada número de un array. Una de las herramientas más poderosas a la hora de trabajar con arrays en javascript: MAP.

const numeros4 = [1,2,3,4,5];

Ejercicio 6

Filtrar números pares de un array. Primera toma de contacto con filter(). 

const numeros5 = [23, 45, 67, 82, 44, 66, 78];

Ejercicio 7

Filtrar palabras de un array con más de tres letras. Revisando el js del repositorio se puede ver que la condición del filter puede ser cualquiera.

const palabras = ["hola", "adiós", "amigo", "sol", "luna", "mar"];
